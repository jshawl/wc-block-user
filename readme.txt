== Description ==

This WooCommerce plugin adds a setting to WooCommerce to block specific user's
email addresses from completing a checkout.

== Frequently Asked Questions ==

= Why would I install this plugin? =

In order to block certain email addresses from checking out in a WooCommerce shop.
